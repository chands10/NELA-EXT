# NELA

Flask application for the NELA Toolkit.

## Requirements

* Ensure you have the latest version of pip or pip3 via:
    * `pip install --upgrade pip`

* Install the following python packages using `pip install`
    * flask
    * flask-wtf
    * flask-bootstrap
    * sqlalchemy
    * psycopg2

* Run the flask app using either:
    * `python app.py`
    * `flask run`
